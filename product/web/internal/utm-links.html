How to read this in GA4

Go to GA4 > Reports > Acquisition > Traffic acquisition. The "Session source / medium" column will show each UTM source separately (reddit / social, nextdoor / social, etc.) instead of everything being (direct) / (none).

To see which source drives the most email signups: GA4 > Explore > create a Free Form exploration, put "Session source / medium" as a dimension and "sign_up" as an event metric. You'll see which channel converts best.

UTMs only work if you use the link when posting. The base lv2park.com link won't track the source.
